Contexte

Dans le cadre de mon master 1 MIAGE à l’UPJV j’ai effectué un stage en développement informatique au sein de l'entreprise Clésence.
Durant ce stage, j'étais amené à intervenir sur le système d'informations de l'entreprise afin d'améliorer leurs fonctionnements, ces applications servent à la mise en contact de l'entreprise avec ses clients.

Description de projet

Il s’agissait du développement autour de l’ERP de l’entreprise dans le but de le faire évoluer en ajoutant des nouvelles fonctionnalités.
Ces fonctionnalités sont utilisées par les personnels de l’entreprise dans différents domaines (Comptabilité, Management, gestion locative…).

Tâches effectuées

  • Parametrage de l'ERP de l'entreprise. : Certaines fonctionnalités existait dans l'ERP, j'étais demandé de les paramétrer afin qu'elles retournent les bonnes données.

  • Développement autour de l'ERP de l'entreprise : Il s'agit du développement des fonctionnalités selon les demandes des personnels de l'entreprise dans les différentes fonctions (Comptabilité, Management, gestion locative...).

  • S'adapter au langage de l'entreprise : L'ERP de l'entreprise a été développé par un langage particulier qui s'appelle Uniface, ce langage était nouveau pour moi et manque d'informations sur internet, vers la fin de stage j'étais capable de développer avec ce langage en autonomie.

Technologies utilisés

  • Uniface (Développement Backend autour de l'ERP)

  • Javascript/HTML/CSS (Développement Frontend des sites de l'entreprise)

  • PHP (Développement backend des sites de l'entreprise )

  • Microsoft SQL Server (Base de données)
